# Smart Triage MEOWS

Sistem Smart Triage menggunakan Rule-Based MEOWS (Modified Early Obstetric Warning Score) Scoring Engine untuk membantu tenaga medis menilai tingkat risiko pasien secara cepat dan objektif.

## Struktur Proyek
- `scoring.py` — Engine utama penghitungan skor MEOWS dan klasifikasi risiko pasien.
- `requirements.txt` — Daftar dependency Python.

## Kategori Risiko
- **Low Risk**: skor 1–4
- **Moderate Risk**: skor 5–6
- **High Risk**: skor ≥ 7 atau ada red trigger (parameter dengan skor 3)
